# Projet : MacBook SSD Base Extension

## 1. Noms de code (Propositions)
*   **AuraDock** : Suggère quelque chose qui entoure et complète le Mac.
*   **Stratus** : Évoque le stockage (Cloud/Nuage) mais sous forme de couche physique.
*   **BaseLayer** : Simple, technique, évoque la protection et l'ajout de fonctions.
*   **SolidShell** : Met l'accent sur la robustesse et le SSD.
*   **OmniBase** : Pour l'aspect "tout-en-un".

## 2. Analyse Technique Préliminaire
### Contraintes de taille :
*   **SSD NVMe (M.2)** : Très fin (~2.2mm à 3.5mm), idéal pour garder le Mac svelte.
*   **SSD 2.5"** : Standard 7mm ou 9.5mm. *Note : Intégrer un disque de 7mm ajoutera une épaisseur significative au MacBook (qui fait environ 15-16mm au total).*
*   **Matériau** : Aluminium (pour la dissipation thermique et l'esthétique Apple).

### Connectique :
*   Passage par les ports latéraux via un connecteur USB-C / Thunderbolt ultra-plat "flush" pour minimiser le levier sur le port.
